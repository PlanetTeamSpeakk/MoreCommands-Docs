# Fix Anim Item Seams (`boolean`)
# Default: `false`
Removes transparent lines in animated item models generated from 2D textures.

:::danger Performance-issues
You may encounter lower framerates if you enable this option.
:::
:::danger Restart required
Toggling this option requires a restart to take (full) effect.
:::
