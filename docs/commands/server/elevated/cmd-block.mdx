# Cmd Block Command
# Usage: /cmdblock \[amount] \[player]
Gives a command block item

## Parameters
- **Amount**
+ Type: `int`
+ Default: `1`
+ The amount of command blocks to give
- **Player**
+ Type: `player`
+ Default: `command source`
+ The player to give the command blocks to

## Example
`/cmdblock 5 PlanetTeamSpeak`
