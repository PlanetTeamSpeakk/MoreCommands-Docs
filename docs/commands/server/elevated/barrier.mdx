# Barrier Command
# Usage: /barrier \[amount] \[player]
Gives a barrier item

## Parameters
- **Amount**
  + Type: `int`
  + Default: `1`
  + The amount of barriers to give
- **Player**
  + Type: `player`
  + Default: `command source`
  + The player to give the barriers to

## Example
`/barrier 5 PlanetTeamSpeak`
