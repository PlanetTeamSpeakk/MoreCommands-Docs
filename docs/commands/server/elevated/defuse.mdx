# Defuse Command
# Usage: /defuse <range\>|all
Defuse primed TNT entities and drop TNT items in their place.

## Parameters
- **Range**
  + Type: `int`
  + Default: none, this parameter is required
  + The range around the command source to defuse all TNT entities in, or `all` to defuse all TNT entities.

## Example
`/defuse 25`
