# Light Command
# Usage: /light \[level] \[count]
Gives a light block item.

## Parameters
- **Level**
  + Type: `int`
  + Default: `15`
  + The light level of the light block
- **Count**
  + Type: `int`
  + Default: `1`
  + The amount of light blocks to give

## Example
`/light 15 1`

:::danger 1.17+
Light blocks were added in 1.17, so this command can only be used on 1.17 and above.
:::
