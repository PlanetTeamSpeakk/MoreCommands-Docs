# Fireball Command
# Usage: /fireball \[power] \[speed] \[impacts]
Shoots out a fireball from your face.

## Parameters
- **Power**
  + Type: `float`
  + Default: `4.0`
  + The power with which the fireball should explode
- **Speed**
  + Type: `double`
  + Default: `1.0`
  + The speed at which the fireball should go
- **Impacts**
  + Type: `int`
  + Default: `1`
  + The amount of times the fireball should explode before disappearing, use -1 for infinite

## Example
`/fireball 4 1 1`
