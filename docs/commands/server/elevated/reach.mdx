# Reach Command
# Usage: /reach \[reach]
Set your reach

## Parameters
- **Reach**
  + Type: `double`
  + Default: none, not passing this displays your current reach
  + 

## Example
`/reach 16`

:::danger Server-only mode
This command is not available in [server-only mode](../../../server-only) as it requires client-side modifications.
:::
