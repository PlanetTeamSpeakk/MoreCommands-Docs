# Gamerule Desc Command
# Usage: /gamerule <gamerule\> desc
Gives the description for the given gamerule. If the gamerule has no description, 
a translation key will be displayed instead.

## Parameters
- **Gamerule**
  + Type: `gamerule`
  + Default: none, this parameter is required
  + The gamerule whose description to display

## Example
`/gamerule vaultRows desc`
