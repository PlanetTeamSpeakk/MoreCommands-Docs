# Spawn Mob Command
# Usage: /spawnmob <entity\> \[amount]
Spawns the given amount of the given entity wherever you're looking.

## Parameters
- **Entity**
  + Type: `entity type`
  + Default: none, this parameter is required
  + The type of entity to spawn
- **Amount**
  + Type: `int`
  + Default: `1`
  + The amount of entities to spawn

## Example
`/spawnmob minecraft:ghast 10`
