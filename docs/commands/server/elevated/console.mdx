# Console Command
# Usage: /console <cmd\>
Run a command as the console

## Parameters
- **Param**
  + Type: `command`
  + Default: none, this parameter is required
  + The command to execute

## Example
`/console say Hello`

:::danger Caution
This command can run any other command as the server, regardless of whether the command source
has permission to run those commands.  
**Grant with caution**
:::
