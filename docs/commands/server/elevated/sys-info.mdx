# Sys Info Command
# Usage: /sysinfo
View system information.

## Parameters
This command has no parameters.

## Example
`/sysinfo`

:::tip
On singleplayer, use the [C Sys Info Command](../../client/c-sys-info) instead.
:::

:::danger Sensitive information
This command contains potentially sensitive information you may not want regular player to 
be able to see, be careful when giving people permission for this.
:::
