# Dimension Command
# Usage: /dimension <dimension\> \[player]
Teleport to a dimension  
Effectively shorthand for `/execute in <dimension> run teleport @s ~ ~ ~`

## Parameters
- **Dimension**
  + Type: `dimension`
  + Default: none, this parameter is required
  + The dimension to teleport to
- **Player**
  + Type: `player`
  + Default: `command sender`
  + The player to teleport

## Example
`/dimension minecraft:the_end PlanetTeamSpeak`
