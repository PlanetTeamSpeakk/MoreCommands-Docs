# Rename Command
# Usage: /rename <name\>
Renames the item you're holding.

## Parameters
- **name**
  + Type: `string`
  + Default: none, this parameter is required
  + The new name of the item, may contain formattings such as `&6`, `&c`, `&l`, etc.

## Example
`/rename &1&l&k||&9&lAerondight&1&l&k||`
