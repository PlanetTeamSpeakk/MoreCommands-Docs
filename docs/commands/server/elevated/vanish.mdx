# Vanish Command
# Usage: /vanish \[players]
Toggles either your vanish or the given players' vanish.

## Parameters
- **Players**
  + Type: `player selector`
  + Default: `@s (command source)`
  + The players whose vanish to toggle.  
    Passing this requires the `morecommands.vanish.others` permission on version 4.0+. 

## Example
`/vanish @a`
