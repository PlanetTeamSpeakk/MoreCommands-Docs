# Broadcast Command
# Usage: /broadcast <msg\>
Send a message to the entire server.  
Effectively the same as `/tellraw @a <msg>`, but uses normal text instead of JSON.  
You can use formattings such as `&6`, `&c`, `&l`, etc.

## Parameters
- **Msg**
  + Type: `string`
  + Default: none, this parameter is required
  + The message to send, can use formattings

## Example
`/broadcast This message will be seen by everyone, &6and this is gold&r!`
