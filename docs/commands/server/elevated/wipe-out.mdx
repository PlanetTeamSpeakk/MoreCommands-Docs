# Wipe Out Command
# Usage: /wipeout \[all|monsters|mobs|animals|living|player|other]
[Removes](./remove) either any entity that's not a player or any entity in the given group.

## Parameters
- **Group**
  + Type: `entity group`
  + Default: `anything but players`
  + The group of entity types to remove.

## Example
`/wipeout monsters`
