# Damage Command
# Usage: /damage <entities\> <source\> <damage\>
Deal damage to anything and anyone.

## Parameters
- **Entities**
  + Type: `entity selector`
  + Default: none, this parameter is required
  + The entities to damage
- **Source**
  + Type: `damage source`
  + Default: none, this parameter is required
  + The source the damage comes from
- **Damage**
  + Type: `float`
  + Default: none, this parameter is required
  + The amount of damage to deal

## Example
`/damage @a[type=!player,distance=..25] outOfWorld 100`
