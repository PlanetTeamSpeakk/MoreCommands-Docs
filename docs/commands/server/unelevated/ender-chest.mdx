# Ender Chest Command
# Usage: /enderchest \[player]
Opens your enderchest from anywhere.

## Parameters
- **Player**
  + Type: `player`
  + Default: `@s (command source)`
  + The player whose enderchest to open.  
    Passing this requires the `morecommands.enderchest.others` permission.

## Example
`/enderchest PlanetTeamSpeak`
