# Home Command
# Usage: /home \[home]
Teleports you to your home.

## Parameters
- **Home**
  + Type: `string`
  + Default: `'home'`
  + The home to teleport to

## Example
`/home home`

## Homes Command
### Usage: /homes
Lists your homes

### Parameters
This command has no parameters.

### Example
`/homes`

## Set Home Command
### Usage: /sethome \[name]
Sets a home where you're currently standing.

### Parameters
- **Name**
  + Type: `string`
  + Default: `'home'`
  + The name of the home to set.  
    The maximum number of homes a player may set is dictated by the [`maxHomes` rule](/gamerules/max-homes).

### Example
`/sethome home`

## Del Home Command
### Usage: /delhome <home\>
Deletes a home.

### Parameters
- **Home**
  + Type: `string`
  + Default: none, this parameter is required
  + The name of the home to delete

### Example
`/delhome home`
