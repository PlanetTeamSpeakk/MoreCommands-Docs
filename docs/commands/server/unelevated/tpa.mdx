# Tpa Command
# Usage: /tpa <player\>
Sends a request to teleport to a player.

## Parameters
- **Player**
  + Type: `player`
  + Default: none, this parameter is required
  + The player to send the request to

## Example
`/tpa PlanetTeamSpeak`

## Tpa Here Command
### Usage: /tpahere <player\>
Sends a request to teleport to you to a player.

### Parameters
- **Player**
+ Type: `player`
+ Default: none, this parameter is required
+ The player to send the request to

## Tp Accept Command
### Usage: /tpaccept
Accepts a teleportation request.

### Parameters
This command has no parameters

### Example:
`/tpaccept`

## Tp Deny Command
### Usage: /tpdeny
Denies a teleportation request.

### Parameters
This command has no parameters

### Example:
`/tpdeny`
