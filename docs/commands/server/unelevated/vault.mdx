# Vault Command
# Usage: /vault <vault\> \[player]
Opens a vault.

## Parameters
- **Vault**
  + Type: `int`
  + Default: none, this parameter is required
  + The number of the vault to open
- **Player**
  + Type: `player`
  + Default: `@s (command sender)`
  + The player whose vault to open  
    Passing this requires op (all versions) or the `morecommands.vault.others` permission (4.0+)

## Example
`/vault 1`
