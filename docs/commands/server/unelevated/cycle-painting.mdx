# Cycle Painting Command
# Usage: /cyclepainting \[motive]
Sets the motive of the painting you're looking at to either the next motive in line, 
or the passed motive.

## Parameters
- **Motive**
  + Type: `painting motive`
  + Default: `next motive in line`
  + The new motive to set for the targeted painting

## Example
`/cyclepainting minecraft:aztec`
