# Map Pic Command
# Usage: /mappic
Takes a picture of either the map you're holding or the map you're looking at and saves it
under `<minecraft directory>/maps/`.

## Parameters
This command has no parameters.

## Example
`/mappic`

## Map Pic Stitch Command
### Usage: /mappic stitch
Stitches multiple maps together into one image. The corners must both be set first.

### Parameters
This command has no parameters.

### Example
`/mappic stitch`

## Map Pic Stitch Corner Command
### Usage: /mappic stitch c1 OR /mappic stitch c2
Sets the corners of the map item frames to stitch together.  
Any map in this rectangle will be stitched together.

### Parameters
These commands have no parameters.

### Example
`/mappic stitch c1`
