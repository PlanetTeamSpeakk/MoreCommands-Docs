# Join Notify Command
# Usage: /joinnotify <player\>
Toggles the join notification for a given player.  
This is separate from vanilla's join notification and is displayed both in chat and on your 
action bar.  

It is just a simple message saying that this player has joined or left.  

By default, this is enabled for all your friends on NameMC. This can be disabled using the [`joinNotifyNameMC` client option](/client-options/tweaks/join-notify-namemc).

## Parameters
- **Player**
  + Type: `player`
  + Default: none, this parameter is required
  + The player to toggle join notifications for

## Example
`/joinnotify PlanetTeamSpeak`
