# C Score Command
# Usage: /cscore teams|team|objectives <team\> \[showMembers]
Read-only client-sided scoreboard command.  
Can list teams you are a part of, display information on a team and list all known objectives.

## Parameters
- **Team**
  + Type: `team`
  + Default: none, this parameter is required
  + The team to show info on, only required with the `/cscore team` command.
- **Show Members**
  + Type: `boolean`
  + Default: false
  + Whether to also list all members of said team.

## Example
`/cscore team goats false`
