# Define Command
# Usage: /define <language\> <query\>
Search up a definition using Google Dictionary (via https://dictionaryapi.dev/).

## Parameters
- **Language**
  + Type: `language`
  + Default: none, this parameter is required
  + The language to look for a definition in  
    Currently supported languages are: `en`, `hi`, `es`, `fr`, `ja`, `ru`, `de`, `it`, `ko`, 
    `pt-BR`, `ar` and `tr`.
- **Query**
  + Type: `string`
  + Default: none, this parameter is required
  + The query to look up

## Example
`/define en synonym`
