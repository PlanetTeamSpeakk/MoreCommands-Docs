# Rotate Command
# Usage: /rotate <yaw\> <pitch\>
Sets your rotation.

## Parameters
- **Yaw**
  + Type: `float`
  + Default: none, this parameter is required
  + The yaw (horizontal rotation) to set
- **Pitch**
  + Type: `float`
  + Default: none, this parameter is required
  + The pitch (vertical rotation) to set

## Example
`/rotate 0 30`
