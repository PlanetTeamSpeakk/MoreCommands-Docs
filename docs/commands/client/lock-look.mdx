# Lock Look Command
# Usage: /locklook
Locks your eyes onto the entity you're currently looking at.

## Parameters
This command has no parameters. 

## Example
`/locklook`

## Lock Look Smooth Command
### Usage: /locklook smooth
Locks your eyes onto the entity you're currently looking at, but just a bit more smoothly.

### Parameters
This command has no parameters.

### Example
`/locklook smooth`
